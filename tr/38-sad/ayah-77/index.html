    ḳâle faḫruc minhâ feinneke racîm.

    Diyanet Isleri:

    Allah: "Defol oradan, sen artık kovulmuş birisin. Din (kıyamet/ceza) gününe kadar lanetim senin üzerinedir" dedi.

    Çık git buradan hemen demişti, gerçekten de taşlanmışsın sen.
